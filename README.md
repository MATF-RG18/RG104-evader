# RG104-evader
Evader
Cilj je igre je sto duze izdrzati a ne sudariti se sa objektima koji nadolaze. Ukoliko se naidje na crveni zid, probija se tako sto se ubrza, zuti zid se izbegava, a kroz zeleni moze da se prodje.

Kontrole:
Start - ENTER
Skok - w
Spustanje - s
Levo, desno - a,d
Ubrzanje - SPACE
